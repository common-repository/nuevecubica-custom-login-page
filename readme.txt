== Description ==

This plugin makes an easy way to customize the  WordPress Login page, using a form with a visual representation of the login page. elements to be modified are: Custom link text, Custom URL, Custom Image Logo, , Custom Message, Custom Message color, Custom color background, Custom image background, Custom form background, Custom label color, Custom input color, Custom input background, Custom form border-radius, Custom form box-shadow, Custom link color, Custom link background, Custom link padding, Custom link border-radius

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the left custom login link to configure the plugin
